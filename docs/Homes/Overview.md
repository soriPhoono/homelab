Our architecture allows for **Home Specialization** across different systems. This means a user profile can be tailored for:

- **Laptops**: Full desktop apps and GUI tools.
- **Droids**: CLI-focused mobile environments.
- **Servers**: Minimal, remote-access oriented setups.

Configurations are stored in `homes/` and associated with systems via the `users` option.
